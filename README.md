# Ferro IG Publisher

Last updated: 2026-02-27

Ferro IG Publisher is a private automation app used to publish user-approved image content to Instagram through Meta Graph API.

---

## Privacy Policy

### What data we access
- Basic Facebook profile data needed for login/authentication
- Page and Instagram account identifiers required for account linking
- Access tokens/permissions required to publish media on behalf of the authorized user

### How we use data
- Authenticate the account owner
- Identify the correct connected Instagram professional account
- Publish media content that the user explicitly requests

### Data retention
- Tokens/configuration may be stored securely only as needed to operate publishing workflows
- We do not sell personal data

### Data sharing
- Data is not sold or shared for advertising
- Data is only used with Meta APIs to perform requested publishing actions
### User controls
- You can revoke app access anytime in Facebook/Instagram settings

For privacy requests or questions: **you@example.com**
